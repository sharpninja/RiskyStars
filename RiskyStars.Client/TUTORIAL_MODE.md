# Tutorial Mode

Tutorial Mode is a guided single-player onboarding path launched from the main menu.

## Launch Flow

1. Click **Tutorial Mode** from the main menu.
2. The client starts an embedded single-player server.
3. A canned scenario is created with player **Cadet** against one easy AI opponent, **Drill Marshal Vega**.
4. The game opens directly into the map with the guided tutorial panel visible.

## Guided Steps

The tutorial panel walks through the core loop:

1. Wait for the world snapshot.
2. Read the turn and phase banner.
3. Select a map object.
4. Open the shortcut sheet.
5. Resolve production.
6. Open the player dashboard.
7. Buy a starter army.
8. Advance to reinforcement.
9. Select a reinforcement target.
10. Advance to movement.
11. Select a mobile army.
12. Issue or inspect movement.
13. Open reference help.
14. Continue into free play.

## Controls

- **Back** returns to the previous tutorial step.
- **Next** advances after a completed or manual step.
- **Skip** advances before the current objective is detected.
- **Full Guide** opens the existing F6 tutorial lesson browser.
- **End** closes tutorial mode and leaves the current game running.

Tutorial Mode does not save step progress. Each launch starts from step 1.
